# MINIMI handmade

Односторінковий інтернет-магазин пледів та кардиганів ручної роботи. Next.js 16 + Tailwind CSS 4.

## Запуск

```bash
npm install
npm run dev
```

## Корисне

- Товари редагуються у `src/lib/products.ts`
- Фото товарів потрібно покласти у `public/products/` (напр. `pled-1.jpg`, `cardigan-1.jpg`)
- Кошик зберігається у localStorage
- Кольори бренду — у `src/app/globals.css`

## Деплой

Підключіть репозиторій до Vercel — збірка відбувається автоматично.
